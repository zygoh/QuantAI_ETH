## 📁 项目结构

```
eth-trading-system/
├── backend/                 # 后端服务
│   ├── app/
│   │   ├── api/            # API路由
│   │   ├── core/           # 核心配置
│   │   ├── services/       # 业务服务
│   │   └── utils/          # 工具函数
│   ├── models/             # 模型文件
│   └── logs/               # 日志文件
├── frontend/               # 前端应用
│   ├── src/
│   │   ├── components/     # React组件
│   │   ├── contexts/       # 上下文
│   │   └── services/       # API服务
│   └── public/             # 静态文件
├── monitoring/             # 监控配置
├── docker-compose.yml      # 开发环境
├── docker-compose.prod.yml # 生产环境
├── start.sh               # 启动脚本
├── deploy.sh              # 部署脚本
└── check-system.sh        # 状态检查
```

## 🚀 快速启动

### 开发环境
```bash
# 1. 配置API密钥
nano backend/.env

# 2. 启动系统
./start.sh

# 3. 访问应用
# 前端: http://localhost:3000
# 后端: http://localhost:8000
```

### 生产环境
```bash
# 1. 配置生产环境
nano .env.prod

# 2. 部署系统
./deploy.sh

# 3. 检查状态
./check-system.sh
```
